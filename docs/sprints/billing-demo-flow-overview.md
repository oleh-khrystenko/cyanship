# Billing Demo Flow — Короткий опис

Робимо сторінку Billing, яка працює як живе демо платіжної системи через Stripe у тестовому режимі.

## Що це дає

Відвідувач лендінгу бачить тізер з цінами → переходить на сторінку Billing → підписується або купує кредити через справжній Stripe Checkout (тестовий режим, гроші не списуються). Це демонструє що платіжка реально працює.

## Два типи оплати

1. **Підписка (Subscription)** — щомісячний план "Pro". Юзер підписується, бачить статус, може скасувати через Stripe Portal.
2. **Кредити (Credit Packs)** — одноразова покупка пачок кредитів (5, 10, 20 штук). Баланс кредитів відображається на сторінці.

## Як працює flow

1. Юзер заходить на `/billing` (потрібна авторизація)
2. Бачить демо-банер з тестовою карткою (`4242 4242 4242 4242`)
3. Обирає підписку або кредити → натискає кнопку
4. Відкривається Stripe Checkout → вводить тестову картку → оплачує
5. Повертається на `/billing/success` → бачить оновлений статус
